# Convert any image into x256 color or b/w unicode

## Use-cases
- Make your CLI application pretty!
- Generate art or use univision as a filter

## How To
Color: `./convert.py <path_to_image_file>`

B/W: `./convert.py <path_to_image_file> bw`
